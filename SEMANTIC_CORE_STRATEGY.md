# Стратегия SEO и Семантическое Ядро для Arbarea 2026

## 1. Введение: Критический подход к алгоритмам

Текущие алгоритмы поисковиков (Yandex YP, Google AI Overviews) требуют не просто наличия ключей, а **экспертности, авторитетности и надежности (E-E-A-T)**. Для SPA-приложения (Single Page Application) критически важно устранить технические барьеры для индексации.

## 2. Семантическое Ядро (Semantic Core)

Мы делим запросы на 3 кластера для максимального охвата.

### Кластер A: Высокочастотные (Брендинг и Категории)

_Цель: Охват верхней части воронки._

- `мебель из массива дуба`
- `авторская столярная мастерская`
- `эксклюзивная мебель из дерева`
- `лофт мебель на заказ`
- `эко декор для дома`

### Кластер B: Товарные (Транзакционные)

_Цель: Прямые продажи. Самая высокая конверсия._

- `купить деревянное панно на стену`
- `рейлинги для кухни из дерева купить`
- `держатель для полотенец дуб`
- `подставка для телефона из массива`
- `деревянный декор для ванной`

### Кластер C: Информационные (Блог/Экспертность)

_Цель: Привлечение трафика через ответы на вопросы._

- `как ухаживать за мебелью из дуба`
- `дуб или ясень что лучше для мебели`
- `идеи декора из дерева в интерьере`

## 3. Техническая Стратегия (Technical SEO)

### 3.1. Управление мета-данными (React Helmet)

Поисковые роботы (особенно Яндекс) могут плохо индексировать динамический JS контент, если мета-теги не меняются.
**Решение:** Внедрить динамическую генерацию `<title>`, `<meta description>`, `<link rel="canonical">` на каждой странице товара и категории.

### 3.2. Микроразметка Schema.org (JSON-LD)

Это "язык", который понимают роботы. Мы должны "скормить" им структуру данных, чтобы получить расширенные сниппеты (цену, наличие, рейтинг) в выдаче.
**Типы:**

- `Product`: На странице товара (цена, валюта, наличие).
- `BreadcrumbList`: Хлебные крошки (структура сайта).
- `Organization`: На главной (лого, соцсети).

### 3.3. Скорость и PWA

Google учитывает Core Web Vitals.

- **LCP (Largest Contentful Paint):** Оптимизация изображений (WebP - уже сделано).
- **CLS (Cumulative Layout Shift):** Фиксация размеров блоков (скелетоны при загрузке).

## 4. Алгоритмический подход к SEO (Implemented)

Мы внедрили самообучающуюся систему мета-данных, которая адаптируется под контент.

### 4.1. Динамическая LSI-инъекция

Компонент `SEO.jsx` теперь использует алгоритм, который автоматически подмешивает сопутствующие ключевые слова (LSI) на основе категории и материала товара.

### 4.2. Knowledge Graph Ingestion (JSON-LD)

Каждая страница теперь "скармливает" поисковикам структурированные данные в 4 слоях:

1. **Organization/LocalBusiness**: Повышает доверие к бренду Arbarea.
2. **BreadcrumbList**: Создает цепочки навигации в поиске.
3. **Product Schema**: Показывает цену, наличие и отзывы.
4. **FAQ Schema**: Захватывает дополнительные блоки в выдаче через ответы на вопросы.
